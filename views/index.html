Проект - Чат

Автор Александр Беликов

Приложение представляет собой комнату, где различные люди могут обмениваться сообщениями в режиме реального времени. Помимо текста, вы можете обмениваться изображениями и Emojies. Для использование приложения необходимо быть зарегестрированным пользователем. Если у вас нет аккаунта, то вы можете создать его по кнопке внизу.

Данное приложение использует:

  • NodeJS и ExpressJS для реализации сервера.

  • Socket.IO для работы многопоточного чата

  • MongoDB для хранения базы данных пользователей

  • Vanilla HTML & CSS для реализации интерфейса
